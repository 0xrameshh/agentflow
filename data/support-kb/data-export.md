# Data Export and GDPR Compliance

FlowDesk supports full data exports for backup, migration, or GDPR data portability requests.

## Standard export
See account-settings.md for basic export instructions.

**Delivery times by export size:**

| Export Size | Estimated Time |
|-------------|---------------|
| < 100 MB | Within 15 minutes |
| 100 MB – 1 GB | Within 2 hours |
| 1 GB – 10 GB | Within 24 hours |
| > 10 GB | Within 7 days |

Large exports (>1GB) are delivered as a **zip archive via secure download link**. The link expires after 7 days.

## GDPR data subject requests

### Access request (Article 15)
To request all personal data FlowDesk holds about you:
1. Go to **Settings → Data → GDPR Requests**.
2. Select **Access Request**.
3. Submit. We fulfill requests within **30 calendar days**.
4. The report is delivered as a machine-readable JSON file to your registered email.

### Deletion request (Article 17 — Right to be forgotten)
1. Go to **Settings → Data → GDPR Requests**.
2. Select **Deletion Request**.
3. Confirm by entering your account email and typing "CONFIRM DELETE".
4. Data is erased within **30 calendar days**.
5. Anonymized aggregate data (e.g., usage statistics) may be retained for analytics.

### Data portability (Article 20)
1. Go to **Settings → Data → GDPR Requests**.
2. Select **Portability Request**.
3. Choose data scope: Tasks, Comments, Projects, or All.
4. Choose format: **CSV** (recommended for spreadsheets) or **JSON** (recommended for migration).
5. Delivery within **30 calendar days**.

## Enterprise-managed exports
Enterprise customers can request a **bulk workspace export** via their account manager. Includes all project data, attachments, and audit logs.

## FAQ
**Q: Can I export only specific projects?**
A: Yes — select individual projects during the export request.

**Q: How long are exports stored?**
A: Exported files are stored for 7 days. Request a new export if yours expires.

**Q: Is the export encrypted?**
A: Yes — all export downloads are served over HTTPS. ZIP files for large exports are password-protected (password sent in a separate email).

For billing-related exports, see billing-faq.md.
